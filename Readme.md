# 🚕 Taxi RL — инструкция для новичка

Привет! Это проект, где компьютер сам учится управлять такси. Он учится сажать пассажира и отвозить его куда нужно. Всё происходит в игре Taxi-v3.

---

## 🧰 Что нужно установить

Перед запуском нужно установить **Python** и несколько библиотек. Не пугайся, это просто.

### 1. Python

- Зайди на сайт `python.org`
- Скачай версию **3.10** или **3.11** (не 3.12 — с ней могут быть проблемы)
- При установке **обязательно** поставь галочку "Add Python to PATH"

После установки открой **терминал** (на Windows — cmd или PowerShell, на Mac/Linux — просто терминал) и проверь:

```bash
python --version
```

Должно показать что-то типа `Python 3.10.x`

---

## 📦 Библиотеки

Это специальные кирпичики, из которых строится программа. Установи их одной командой:

```bash
pip install gymnasium numpy matplotlib streamlit
```

Что каждая делает:
- **gymnasium** — даёт игру Taxi
- **numpy** — помогает с числами
- **matplotlib** — рисует графики
- **streamlit** — создаёт красивый веб-интерфейс

---

## 🚀 Как запустить проект

У тебя будет **два главных файла**:

### Шаг 1 — Обучить агента

Запускаешь в терминале:

```bash
python train.py
```

Компьютер начнёт учиться. Это займёт 1–2 минуты. Ты увидишь, как постепенно улучшается результат. В конце появится график обучения и сохранятся два файла: `q_table.npy` (память агента) и `train_rewards.npy` (история наград).

**Важно:** Дождись полного завершения! Будет написано "Training finished!"

### Шаг 2 — Запустить игру

```bash
streamlit run app.py
```

Откроется браузер с игрой. Там будет:
- 🚕 Такси на карте
- 🧍 Пассажир
- 🏠 Пункт назначения

**Кнопки:**
- **Step** — один шаг (нажимай, чтобы смотреть по шагам)
- **Auto** — автоигра (сам довезёт пассажира)
- **Reset** — начать заново

Справа ты видишь:
- Статистику (шаги, награда, итого)
- График наград за текущую игру
- **График обучения** — как агенту становилось лучше от эпизода к эпизоду
